# Hollow Sanctum – Python Text Adventure Game

## Description

Hollow Sanctum is a text-based adventure game written in Python for an introductory scripting course.
The player explores a haunted castle, collects sacred relics, and attempts to defeat the Hollow King.

The game runs entirely in the terminal and demonstrates core programming concepts such as dictionaries, functions, loops, conditionals, and user input handling.

## Features

* Text-based room navigation (North, South, East, West)
* Inventory system
* Item collection
* Room descriptions
* Boss encounter with win/lose conditions
* Command-based input system

## How to Run

Make sure Python is installed.

Run the game from the project folder:

```bash
python game.py
```

(or whatever your file name is)

## How to Play

Commands:

* Move: `north`, `south`, `east`, `west`
* Pick up item: `get <item name>`
* Exit game: `exit`

Example:

```
north
get Silver Mirror
west
```

## Goal

Collect all required relics before entering the Hollow Sanctum.

Required items:

* Anointed Sword of Dawnlight
* Silver Mirror
* Iron Bell
* Binding Chain of Salt and Twine
* Ashen Candle
* Book of the Blood Sacrifice

If you enter the Hollow Sanctum without all items, the Hollow King will defeat you.

## Concepts Demonstrated

This project demonstrates:

* Python dictionaries for game world structure
* Functions for game logic
* Loops for gameplay
* Lists for inventory management
* String parsing for user commands
* Conditional logic for win/lose outcomes
